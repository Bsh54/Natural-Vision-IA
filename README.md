#  Vision Naturelle - Reconnaissance d'Image par IA

Une application web élégante de reconnaissance d'images utilisant TensorFlow.js et MobileNet avec une interface inspirée de la nature.

![Vision Naturelle Screenshot](https://via.placeholder.com/800x450/8B7355/FFFFFF?text=Vision+Naturelle)

##  Fonctionnalités

- Reconnaissance d'images en temps réel - Utilise le modèle MobileNet pour analyser et classifier les images
-  Interface naturelle - Design organique avec des couleurs terreuses et des animations fluides
-  Responsive - S'adapte à tous les écrans (mobile, tablette, desktop)
-  Pas d'installation requise - Fonctionne directement dans le navigateur
-  Confidentialité - Les images sont traitées localement, aucun envoi serveur

##  Démo en ligne

[Lien vers la démo] -  https://natural-vision-ia.onrender.com/ 

##  Technologies Utilisées

- **TensorFlow.js** - Framework d'IA côté client
- **MobileNet v2** - Modèle de reconnaissance d'images pré-entraîné
